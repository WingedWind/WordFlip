# WordFlip

WordFlip — учебное веб-приложение на Django для создания и повторения флэшкарточек. Пользователь может добавлять карточки с терминами и пояснениями, объединять их в наборы, редактировать их и проходить тренировку по выбранному набору.

## Возможности

- создание новых карточек
- редактирование карточек
- создание нового набора при добавлении карточки
- просмотр всех карточек
- фильтрация карточек по набору
- режим тренировки по выбранному набору
- валидация данных в формах

## Технологии

- Python 3
- Django 6
- SQLite
- HTML
- CSS
- JavaScript

## Запуск проекта

1. Перейти в папку проекта:

```bash
cd WordFlip
```

2. Установить зависимости:

```bash
pip install -r requirements.txt
```

3. Применить миграции:

```bash
python3 manage.py migrate
```

4. Запустить сервер:

```bash
python3 manage.py runserver
```

После запуска проект будет доступен по адресу:

```text
http://127.0.0.1:8000/
```

## Описание работы

Каждая карточка содержит термин, пояснение и принадлежит определённому набору. Пользователь может создавать карточки, группировать их по наборам и повторять материал в тренировочном режиме, где сначала показывается термин, а затем по нажатию — пояснение.
